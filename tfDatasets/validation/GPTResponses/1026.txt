1. Unit testing: This component involves testing of individual units or modules of the software application. It is the first level of testing that focuses on identifying and fixing defects in the modules.2. Integration testing: This component involves testing of the integration of individual modules or units to form a complete system. It tests the interaction between modules and checks if they work together as expected.3. System testing: This component involves testing the entire software system as a whole. It tests the functional and non-functional requirements of the system and checks if it meets the user requirements and specifications.
