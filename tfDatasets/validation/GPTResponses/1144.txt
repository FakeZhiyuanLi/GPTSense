Components that can change the shape of a network include:1. Routers: These devices help to connect multiple networks together and can change the routing paths in a network.2. Switches: These devices help to connect different devices within a network and can change the flow of data packets in a network.3. Firewalls: These devices help to secure a network by monitoring and controlling the flow of network traffic.4. Access Points: These devices are used in wireless networks to connect devices to the network and can change the coverage area of the network.5. Bridges: These devices help to connect two separate networks together and can change the physical structure of a network.6. Hubs: These devices are used to connect multiple devices within a network and can change how the network handles data traffic.7. Modems: These devices are used to connect a network to the internet and can change the way data is transmitted in the network.
