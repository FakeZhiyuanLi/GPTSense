1. Install malicious software: An unauthorized user can install malware or viruses onto the computer's hard drive or system, which can damage or compromise the security of the computer.2. Steal sensitive data: An unauthorized user can access, copy, or steal sensitive data or confidential information stored on the computer's hard drive or network.3. Modify system settings: An unauthorized user can change system configurations, settings, or passwords, which can cause malfunctioning or implementation of additional security risks.4. Plant a keylogger: An unauthorized user can install a keylogger program that records all the keystrokes typed on the keyboard by users, including passwords and confidential login credentials.5. Access network resources: An unauthorized user can gain access to network resources by connecting the computer to a network port and gaining unauthenticated access to other computers or systems connected to the network.
